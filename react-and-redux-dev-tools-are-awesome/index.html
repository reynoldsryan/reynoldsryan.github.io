React and Redux dev tools are awesome!

17 February 2016

Every software engineer knows how useful Chrome's dev tools are. But did you know there are also extremely helpful Chrome extensions that will make it easier to debug React/Redux apps?

You can download the React dev tool at https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=en.

It lets you navigate through your application's component hierarchy and set break points to check each component's state and props at a given point in your code. This makes it much easier to see how one component affects another.

You can download the Redux dev tool at https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd?hl=en.

Also, you need to add it to your middleware when you create your store like so: 

const createStoreWithMiddleware = compose(applyMiddleware(ReduxPromise, reduxRouterMiddleware), window.devToolsExtension ? window.devToolsExtension() : f => f)(createStore);

First off, it's super colorful and fun to use. But more importantly, it lets you see the state of your application at any point in your code. It also keeps track of each action as it's fired off and you can easily undo actions to pinpoint where something is breaking.
