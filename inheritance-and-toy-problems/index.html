Inheritance and Toy Problems

02 December 2015

After the Thanksgiving break, we did a sprint these past couple days on class inheritance. So now I know how to create a superclass and subclasses who inherit methods and properties of the superclass.

We also started doing toy problems this week. Yesterday I wrote code to determine the number of possible moves in a game of rock paper scissors given 'n' number of rounds. Today the problem consisted of returning the first number in a given array that occurs an even amount of times in the array.

And now to spend the last hour before bed finishing up my presentation to the class tomorrow that will bring together my new coding knowledge along with my legal knowledge - a short primer on code and copyright.
