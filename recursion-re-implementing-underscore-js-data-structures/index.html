Recursion, Re-implementing underscore.js, Data Structures

I've only been at MakerSquare for four days and it feels like four weeks. And I mean that in the best possible way!

In the past four days, I've done the following:

  • Learned what recursion is and how to write recursive functions
  • Implemented 15 underscore.js functions
  • Implemented stack and queue using the following four instantiation patterns: functional, functional with shared methods, prototypal and pseudoclassical
  • Utilized pair programming as both driver and navigator for all of the above (I really like coding this way!)
  • Learned about lexical scoping and closure
  • Used github daily to push my code to my repos
  • Learned how to create an algorithm to solve a problem
  • Went over coding best practices
  • Examined function binding and how to use "this"

Phew!
